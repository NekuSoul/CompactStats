CompactStats

##Description

Compact Rainmeter suite.

##Features

- BackgroundColumn
- Clock
- Disks
- Music
- Network
- System
- Title
- Wifiject: Template for a 3D game

##Usage

- Customization options can be found in the SETTINGS section in Settings.ini
- The System skin needs the Afterburner plugin to work correctly
